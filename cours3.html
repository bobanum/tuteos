Un site Web avec Laravel

Les étapes (temporaire)

Les vidéos sont disponibles sur la chaîne Youtube

  1. Installation du plugin

    1. Faire la recherche Form Laravel dans Google.
    2. Suivre le lien vers Laravel Collective (le 2e).
    3. Suivre les instructions de la version 5.4 sauf la première étape.
    4. Ouvrir le fichier /composer.json
    5. Ajouter dans la section 'require' la ligne suivante (avec la virgule) :
      "laravel/tinker": "~1.0",
      "laravelcollective/html":"^5.4.0"
    6. Dans le terminal faire composer update.

  2. Ajouter le lien et la route

    1. Ajouter le lien vers l'action create
    2. Ajouter la route correspondante
    3. Tester et constater l'erreur

  3. Ajouter la contrainte de route

    1. Tester en inversant les routes.
    2. À la fin de la route show ajouter ->where('user', '[0-9]+')

  4. Le formulaire vide

    1. Dans le controller, ajouter l'appel à la view create.
    2. envoyer un user vide
    3. ajouter une valeur par defaut à province
    4. creer la view create
    5. ajouter le titre l'appel à la view form
    6. constater le non-besoin d'envoyer la variable user
    7. créer la view form
    8. y ajouter open et close
    9. Regarder le code html du formulaire vide.

  5. Les premiers champs

    1. Dans la view form, créer le tableau avec le champ nom et le champ province.

  6. Les autres champs

    1. Ajouter les autres champs.
    2. Utiliser le faker dans le controller pour les tests.
    3. Ajouter le bouton submit.

  7. La réception des données

    1. Dans l'action UserController@store...

  8. La sauvegarde du user

    1. Dans l'action UserController@store...

  9. La redirection

    1. Dans l'action UserController@store...
